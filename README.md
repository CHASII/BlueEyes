### 配置说明
[System]  
`work_time`  
护眼时间段,其他时间段不进行操作  

[Time_config]  
`notice_time`  
系统进入护眼时间段时将会有一个提示框，给你一个暂停手头工作的缓冲时间(单位秒)  
`protect_time`  
提示时间到后,系统将进入护眼时间(单位秒)  
`lockworkstation`  
进入护眼时间时间后是否进行锁屏,1=锁屏,0=不锁屏  
`locktaskmgr`  
锁屏期间是否允许调用任务栏(人员可以调用任务栏可以强制关闭护眼程序),1=禁用,0=允许
